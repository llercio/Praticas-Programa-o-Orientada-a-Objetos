# Práticas de Programação Orientada a Objetos

Repositório de exercícios pessoais para praticar e aprender conceitos de Programação Orientada a Objetos (POO).  
O objetivo principal é estudar e experimentar princípios e padrões de OO (classes, encapsulamento, herança, polimorfismo, interfaces/traits, composição, coleções, exceções, testes, etc.) organizando exercícios pequenos e projetos experimentais com commits regulares para acompanhar a evolução.

Visite o repositório: [llercio/Praticas-Programa-o-Orientada-a-Objetos](https://github.com/llercio/Praticas-Programa-o-Orientada-a-Objetos)

## Objetivos
- Aprender e praticar conceitos de PO.
- Fazer pequenos exercícios e projetos experimentais que ilustrem padrões e boas práticas.
- Manter um ritmo de commits para treinar disciplina e acompanhar progresso.
- Documentar decisões, problemas e soluções encontradas em cada exercício.

## Como rodar
Recomenda-se ter o JDK 11+ instalado. Você pode rodar arquivos ou projetos de várias formas:
IDE:
- Use IntelliJ IDEA, Eclipse, Visual Studio, VS Code (com extensões) para importar o diretório como projeto e rodar as classes/módulos com `main`.
